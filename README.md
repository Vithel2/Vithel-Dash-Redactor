📑 Vithel Dash Redactor
Добро пожаловать в Vithel Dash Redactor (Версия Beta 1.5)! Это фанатский 2D-платформер с полноценным встроенным редактором уровней, вдохновленный Geometry Dash.

🎮 Описание игры
Игра позволяет создать свой собственный уровень с нуля прямо внутри программы.

Основные фишки:
4 режима игрока: Куб (Cube), Кораблик (Ship), Волна (Wave) и НЛО (UFO).

Полноценный редактор: Удобная сетка, палитра объектов (блоки, шипы, склоны, финиш, порталы смены режимов, триггеры скорости).

Move-триггеры: Настраиваемая анимация движения блоков по группам (сглаженное или резкое смещение).

Кастомизация: Смена цвета скина игрока прямо в главном меню.

Кастомная музыка: Поддержка своих аудиофайлов для созданных уровней.

🛠 Как установить и запустить игру ( Описанно нейросетью Gemini )
Так как игра написана на языке Python, для её запуска понадобится выполнить два простых шага:

Шаг 1. Установка Python
Скачай и установи Python (версии 3.8 или новее) с официального сайта python.org.

Важно при установке: Обязательно поставь галочку напротив пункта "Add Python to PATH" (Добавить Python в PATH) в самом низу установщика.

Шаг 2. Установка библиотеки Pygame
Игра использует движок Pygame для графики и звука. Его нужно установить через командную строку:

Открой терминал (Командную строку) на компьютере.

Введи следующую команду и нажми Enter:

Bash
pip install pygame
Дождись окончания загрузки.

Шаг 3. Запуск
Просто кликни два раза левой кнопкой мыши по файлу Vithel Dash 1.5 Beta.py (или запусти его через вашу среду разработки/консоль командой python "Vithel Dash 1.5 Beta.py").

🎵 Как добавить свою музыку на уровень?
Создай в папке с игрой папку с названием Звуки (если её ещё нет).

Закинь туда свои музыкальные треки.

Чтобы редактор их увидел, переименуй файлы так, чтобы их названия начинались с букв MFRL (например: MFRL_my_track.mp3, MFRL2.wav, MFRL_horror.ogg).

Внутри редактора нажми клавишу U, чтобы открыть меню выбора музыки и привязать трек к уровню.

🕹 Быстрое управление
В меню: E — Редактор, P — Играть, A / D — Смена скина.

В игре: Пробел / W / ЛКМ — Прыжок/Действие, R — Быстрый рестарт, E — Мгновенный возврат в редактор.

В редакторе: ЛКМ — Ставить блоки, ПКМ — Удалять, A / D — Двигать камеру, E (наведя на объект) — Настройки параметров, R — Поворот объекта.
