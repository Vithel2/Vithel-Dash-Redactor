# Vithel-Dash-Redactor
Vithel Dash Redactor (Beta 1.4) 2D-платформер с полноценным встроенным редактором уровней, вдохновлённый механикой Geometry Dash. Написан на Python с использованием библиотеки Pygame. 

🚀 Основные возможности
4 игровых режима: Куб (Cube), Кораблик (Ship), Волна (Wave) и НЛО (UFO).

Полноценный редактор: Создание уровней в реальном времени с использованием сетки, палитры объектов и горячих клавиш.

Интерактивные элементы: Блоки, шипы, склоны, финишные триггеры, порталы смены режимов и изменения скорости.

Move-триггеры: Возможность настраивать плавное или резкое перемещение групп объектов при прохождении.

Динамический звук: Процедурная генерация звуковых эффектов (прыжки, порталы, смерть) без использования внешних аудиофайлов.

Кастомизация: Выбор цвета скина игрока в главном меню.

Сохранение: Запись и чтение уровней через формат level.json.

🛠️ Управление
В меню:
E — Открыть редактор уровней

P — Играть в сохранённый уровень

A / D — Сменить скин игрока

В редакторе:
ЛКМ — Поставить объект / Выбрать объект в палитре

ПКМ — Удалить объект

A / D — Перемещение камеры

Цифры 1-0, T, H — Быстрый выбор кисти

E (на объекте) — Меню детальных настроек (цвет, группа, параметры движения)

R — Быстрый поворот объекта

S / L — Сохранить / Загрузить уровень

C — Полная очистка карты

P — Тест уровня (переход в игру)

В игре:
Пробел / W / Стрелка вверх / ЛКМ — Действие (прыжок, взлёт, смена вектора)

R — Рестарт уровня

ESC — Вернуться в меню

Для работы игры требуется установленный пакет pygame.
