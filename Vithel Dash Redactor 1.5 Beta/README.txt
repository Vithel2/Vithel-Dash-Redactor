Vithel-Dash-Redactor
Vithel Dash Redactor (Beta 1.4) 2D-платформер с полноценным встроенным редактором уровней, вдохновлённый механикой Geometry Dash. Написан на Python с использованием библиотеки Pygame.

Для работы игры требуется установленный пакет pygame. ( Просто в командной строке пропишите если у вас уже есть Python: pip install pygame )
Не удалять файл: "level" - используется для сохранения вашего уровня!
Файл "level" автоматически создаётся когда бы впервые сохраните ваш уровень.
Свою музыку вы можете добавлять заменяя песни MFRL (1-9). Названия песен не изменять!
